Stap voor stap

1
Maak een gratis GitHub account
Ga naar github.com en klik op Sign up. Kies een gebruikersnaam — die wordt onderdeel van je link.
https://github.com/signup
2
Maak een nieuw repository aan
Klik rechtsboven op + → New repository.

• Repository name: race-day
• Zet op Public
• Vink aan: Add a README file

Klik Create repository.
3
Upload alle 6 bestanden
Klik op Add file → Upload files.

Sleep alle zes bestanden tegelijk naar het uploadvenster. Wacht tot alles klaar is, klik dan op Commit changes.
💡 Zorg dat het bestand index.html heet — niet victor_index.html.
4
Zet GitHub Pages aan
Ga naar Settings → Pages (links in het menu).

Onder Source: kies Deploy from a branch.
Onder Branch: kies main en map / (root).

Klik Save.
5
Na 1-2 minuten is je link klaar
Ververs de Settings → Pages pagina. Je link staat bovenaan:
https://JOUWGEBRUIKERSNAAM.github.io/race-day/
✅ Stuur deze link via WhatsApp naar je familie. Ze hoeven niets te installeren.
